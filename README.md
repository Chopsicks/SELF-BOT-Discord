📝 О проекте
S-B0T — это мощный self‑бот для Discord, написанный на Python. Он предоставляет широкие возможности управления вашим аккаунтом через консоль и красивую веб‑панель. Бот поддерживает:

📩 Автоматический ответ на личные сообщения (простой текст или генерация с помощью ИИ через OpenRouter)

👥 Управление друзьями (просмотр, добавление, удаление, блокировка)

📨 Массовая рассылка сообщений всем друзьям

📁 Управление заявками в друзья (принять/отклонить, авто‑обработка)

💬 Просмотр и отправка сообщений в DM‑чатах через веб‑интерфейс

🛡️ Работа с серверами (просмотр, выход)

🕹️ Консольные команды для быстрого управления

🎨 Адаптивный веб‑интерфейс в стиле Discord

⚠️ Важно: Использование self‑ботов запрещено правилами Discord и может привести к блокировке аккаунта. Проект создан в образовательных целях.

✨ Возможности
Функция	Описание
🖥️ Веб‑панель	Управляйте ботом через браузер (http://127.0.0.1:5000)
🤖 ИИ‑автоответчик	Подключите OpenRouter (DeepSeek, Mistral, GPT и др.) для естественных ответов
⏱️ Простой автоответчик	Задайте текст и задержку ответа
👤 Управление друзьями	Просмотр, удаление, блокировка, разблокировка
📨 Массовая рассылка	Отправьте одно сообщение всем друзьям сразу
📁 Заявки в друзья	Принимайте/отклоняйте входящие заявки, настройте авто‑принятие
💬 Мессенджер	Просматривайте DM‑каналы и общайтесь прямо в панели
🛡️ Серверы	Просматривайте список серверов и покидайте их
🎛️ Консольные команды	Управляйте ботом без открытия браузера
🔐 Безопасность	Токен хранится в файле, поддержка загрузки через форму
📦 Установка
Клонируйте репозиторий

bash
git clone [https://github.com/frizzylow/s-b0t.git](https://github.com/Chopsicks/SELF-BOT-Discord.git)
cd s-b0t
Установите зависимости

bash
pip install discum flask requests
(Опционально) Создайте файл с токеном
Поместите ваш Discord‑токен в файл discord_token.txt в той же папке.
Либо введите токен позже через веб‑панель или консольную команду /token.

Запустите бота

bash
python bot.py
После запуска вы увидите приветствие и ссылку на панель управления:
Flask запущен! Панель управление доступна. /help - справка

🔧 Использование
Веб‑панель
Откройте в браузере: http://127.0.0.1:5000

Разделы панели:

Главная — статус бота, статистика, отправка сообщения по ID

Друзья — список друзей и заблокированных, управление ими

Заявки — входящие заявки в друзья, авто‑обработка

Мессенджер — список DM‑каналов, переписка в реальном времени

Серверы — все ваши серверы с возможностью покинуть

Настройки — статус аккаунта, массовая рассылка, добавление друга, автоответчик, управление токеном

Консольные команды
В консоли, где запущен бот, доступны следующие команды:

Команда	Описание
/start	Запустить бота (если не запущен)
/stop	Остановить бота
/token	Ввести новый токен вручную
/file	Загрузить токен из файла (укажите путь)
/status	Показать текущий статус бота
/debug on	Включить режим отладки (подробные логи)
/debug off	Выключить режим отладки
/debug status	Показать состояние режима отладки
/help	Справка по командам
🤖 Автоответчик
Простой режим
Задайте текст ответа и задержку (0 — сразу, >0 — бот подождёт указанное время без вашей активности в этом чате).

ИИ‑режим (через OpenRouter)
Получите API‑ключ на openrouter.ai

Выберите модель (например, бесплатный @preset/deepseek)

Настройте системный промпт (как должен вести себя ИИ)

Бот будет отвечать только на последнее сообщение в DM, игнорируя историю (чтобы не зацикливаться)
